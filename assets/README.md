# README images

Drop the screenshots the main README references here, using these exact names:

| File | Shown under | Suggested shot |
|------|-------------|----------------|
| `poo.png` | 💩 Poo | pooping / a poo on the ground with the brown bar segment |
| `toilet-paper.png` | 🧻 Dirty & Toilet Paper | holding the roll / the grey Dirty segment |
| `pee.png` | 💦 Pee | mid-stream with the pee puddle and yellow bar segment |

The main README links to these by absolute URL
(`https://raw.githubusercontent.com/IAteSpaghetti-PEAK/INeedToPEEak/main/assets/<name>.png`)
so they render on both GitHub and the Thunderstore mod page.

Keep files reasonably small (a 1–2 MB PNG or JPG is plenty). If you use `.jpg`
instead of `.png`, update the extensions in the main README to match.
